# 🧠 얼굴 인식 시스템

이 프로젝트는 **Streamlit + FastAPI** 기반 얼굴 인식 시스템입니다.  
현재는 **ResNet18** 모델로 이미지 분류 결과를 반환하며,  
향후 **실시간 얼굴 인식 기능**으로 확장할 예정입니다.
